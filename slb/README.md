# server load balancer
```
负载均衡（主要包括L4/L7），用于提升集群服务的稳定性和处理能力，并能更安全的控制后端服务的上线等等。
```

## L4负载均衡
```
主要对TCP/UDP的流量做负载均衡，方案如LVS、等价路由

这两种方案可以互补
LVS优势:
    工作模式很丰富，能满足大部分需求
    对后端服务透明，达到高性能、高可用，且管理方便
    可以做session同步
等价路由优势:
    在路由器/三层交换机级别做到（转发规则简单，可靠）
```

## L7负载均衡
```
主要对诸如HTTP等L7协议做负载均衡，常用方案如Haproxy,nginx,envoy等
```
