## 结构型模式
```
主要关注类和对象的组合，以解决软件系统的结构问题。它们提供了一种方法来把类或对象组合成更复杂的结构，以解决更加复杂的业务问题。
```

## 行为型模式
```
主要关注对象之间的通信和责任分配，以解决软件系统中的行为问题。它们提供了一种方法来把对象之间的联系和交互规范化，以解决复杂的业务问题。
```
